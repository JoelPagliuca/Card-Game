# Card Game

Python code for playing card games. Not a heap of functionality so far.

## Depencies
* `python2`

## Running
`python main.py`

## Development
### Dependencies
* `virtualenv`
* `makefile` - optional

### Setup
`make dev`

### Unit tests
`make test`
